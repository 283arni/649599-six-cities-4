# Проект "6 городов"
---
### Проект содержит:
  - HTML
  - CSS
  - JavaScript
  - TypeScript для типизации
  - React
  - Redux
  - React-router
  - [leaflet для карты](https://leafletjs.com/)
---
### Запуск проекта:
  1. Устанавливаем Node.js
  2. Клонируем репозиторий git clone "ссылка репозитория" или просто скачиваем архив.
  3. npm i
  4. npm start для запуска проекта
